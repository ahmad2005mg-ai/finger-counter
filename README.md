# finger-counter
A finger counting model (1-5) trained using Google Teachable Machine.

🎯 Project Classes
The model is trained on **5 distinct classes**:
* **Class 1:** 1 Finger ☝️
* **Class 2:** 2 Fingers ✌️
* **Class 3:** 3 Fingers 🤟
* **Class 4:** 4 Fingers 🖐️
* **Class 5:** 5 Fingers ✋

---

## 🛠️ How it was built
1. **Data Collection:** Captured images for 5 different hand gestures.
2. **Training:** Trained the model using Google's Teachable Machine.
